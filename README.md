# CGPA Calculator & Future Planner

A fully client-side CGPA calculator and academic planning tool built for IITM BS students to calculate current CGPA, predict ongoing semester results, and simulate future performance.

This project was created for the Namdapha Tech Challenge with a strong focus on correct grading logic, clean UX, responsiveness, and high-quality TypeScript architecture.
